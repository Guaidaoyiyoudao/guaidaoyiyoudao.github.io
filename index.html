文件目录解释

App

该目录包含了应用的核心代码，大多数类都在这个目录里面。

Bootstrap

Bootstrap里面的app.php用来启动框架。
cache目录包含性能优化的文件。

Config

应用所有的配置文件都在该目录，需要熟悉里面的选项。

Database

包含了factories，migrations,seeds,如果想使用sqlite数据库，可以把文件放在这个目录里面。

Public

里面index.php是所有应用请求和自动加载配置的入口点。这个目录也包括了图片，css,以及javascript。
